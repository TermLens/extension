# TOS Analyzer (서비스 약관 분석기)

복잡하고 어려운 서비스 이용 약관을 AI 서버로 전송하여 분석한 뒤, 사용자에게 종합 평가 등급과 핵심 조항 요약을 시각적으로 제공하는 브라우저 확장 프로그램입니다.

사용자들은 '약관에 동의했다'는 이유만으로 자신도 모르게 개인정보가 판매되거나 불리한 조항을 감수해야 하는 피해를 입곤 합니다. 이 프로젝트는 이러한 정보 비대칭 문제를 해결하고, 사용자가 자신의 권리를 명확히 인지할 수 있도록 돕는 것을 목표로 합니다.

## 주요 기능

* **약관 분석 및 요약:** 현재 페이지의 약관을 분석하여 종합 평가 등급(A~E)을 제공합니다.
* **핵심 조항 시각화:** 사용자에게 유리(`good`), 불리(`bad`), 중립(`neutral`)적인 주요 조항을 아이콘과 함께 요약하여 보여줍니다.
* **필수 항목 자동 동의:** 회원가입 등에서 '필수' 항목은 자동으로 체크하여 편의성을 높입니다 (단, '전체 동의' 항목은 제외).
* **직관적인 UI:** 로딩, 분석 결과, 오류 상태를 명확하게 구분하는 모달 UI를 제공합니다.

## 설치 및 사용 방법

본 확장 프로그램은 Chrome 웹 스토어에 등록되지 않은 '개발자 모드'용 확장 프로그램입니다.

### 1. 설치 방법

1.  본 프로젝트를 로컬에 `git clone` 받거나 ZIP 파일로 다운로드합니다.
2.  Chrome 브라우저에서 `chrome://extensions` 페이지로 이동합니다.
3.  우측 상단의 **'개발자 모드'** 스위치를 켭니다.
4.  좌측 상단에 나타나는 **'압축 해제된 확장 프로그램을 로드합니다'** 버튼을 클릭합니다.
5.  파일 탐색기에서 `manifest.json` 파일이 위치한 프로젝트 폴더를 선택합니다.
6.  설치가 완료되면 브라우저 툴바(주소창 우측)에 아이콘이 나타납니다.


### 2. 사용 방법

1.  분석을 원하는 서비스의 약관 페이지 (예: 회원가입 페이지)에 방문합니다.
2.  브라우저 툴바의 **TOS Analyzer 아이콘**을 클릭합니다.
3.  클릭 즉시 '약관 분석 중...'이라는 로딩 모달이 나타납니다.
4.  잠시 후, 분석이 완료되면 종합 평가 등급과 주요 조항이 요약된 결과 창이 나타납니다.
5.  결과 확인 후, (X) 버튼이나 모달 바깥의 어두운 영역을 클릭하여 창을 닫을 수 있습니다.
